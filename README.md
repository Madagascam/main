# HiChess — AI-Powered Chess Highlights Generator

> Автоматизированная система для анализа шахматных партий и создания динамичных видео с ключевыми моментами игры

## 📌 Описание проекта

HiChess — интеллектуальный сервис, разработанный в рамках инициативы «Цифровая кафедра», который автоматически анализирует шахматные партии и выделяет наиболее драматичные и тактически значимые моменты для создания коротких видео-хайлайтов. Система устраняет рутинный анализ партий и позволяет зрителям и игрокам мгновенно погружаться в самые захватывающие эпизоды игры.

### Ключевые возможности
- Автоматическое определение «интересных» моментов партии с помощью ML-моделей
- Генерация видео с динамичными переходами и визуальными эффектами
- Анализ партии на основе шахматных метрик и поведенческих паттернов
- Интуитивный веб-интерфейс для загрузки партий и просмотра результатов

## 🔬 Подходы к определению «интересных» моментов

В ходе исследования было проанализировано несколько методологий:

| Подход | Преимущества | Ограничения |
|--------|--------------|-------------|
| **Эвристический анализ** | Высокая интерпретируемость, понятные критерии (резкие изменения оценки, тактические удары) | Требует ручной настройки правил |
| **Кластеризация эмбеддингов** | Обнаружение неочевидных паттернов без разметки | Сложность выделения «интересных» кластеров |
| **Трансформер-архитектуры** | Минимальная подготовка данных, работа с контекстом партии | Требует больших объемов данных для обучения |
| **Ансамбль моделей** | Баланс между интерпретируемостью и качеством | Усложнение архитектуры |

*Текущая реализация использует гибридный подход: комбинация шахматных эвристик (изменение оценки позиции, комбинации, цугцванг) с легковесной нейросетью для финальной классификации.*

## 🎨 Дизайн и пользовательский опыт
